Our system uses a two-layer approach for optimal performance and accuracy:

Detection Flow

Escalation Check
Pattern matching for agent requests (instant response)

Dynamic conversation flow execution

🛠️ API Documentation

Core Endpoints

🔗 Connection Details

POST /api/connection_details

Get LiveKit connection details for voice session

💬 Process Message

POST /api/process_flow_message

Process user message through intelligent flow system

📊 Update Session

POST /api/sessions/update

Update session information and state

🧹 Cleanup Room

DELETE /api/rooms/{room_name}

Clean up room resources and session data

Request/Response Example

// Process Flow Message Request POST /api/process_flow_message { "room_name": "session_123", "user_message": "I need pricing information", "conversation_history": [...] } // Response { "status": "processed", "flow_result": { "type": "flow_started", "flow_name": "Pricing", "response": "How many phone lines do you need?" } }
